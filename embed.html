Embed code
HTML 
<iframe
  src="https://yourusername.github.io/prime-today/widget.html"
  width="240"
  height="220"
  frameborder="0"
  scrolling="no"
  title="Prime Today Widget"
  style="border-radius:12px;overflow:hidden;"
></iframe>
Notes
01
Replace yourusername in the src URL with your actual GitHub username before sharing the embed code.
02
The widget uses the visitor's local date - so users in different timezones may see different answers around midnight. This is intentional.
03
The widget loads the DM Mono and DM Serif Display fonts from Google Fonts. If your site already loads them, there's no extra cost.
04
You can resize freely - the widget is responsive within its container. Minimum recommended size is 200 × 180 px.
